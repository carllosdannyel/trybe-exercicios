Testando o Bootstrap

Para se utilizar o bootstrap primeiramente deve-se baxar-lo no site, descompactar e utilizar a versao minima no head da pagina html passando o seu caminho atraves do link, ele precisa da biblioteca jquery pra funcionar, por isso utilizamos na seguinte ordem, primeiramente linkando o caminho do css minimo, o jquery e por fim o js minimo que esta dentro do bootstrap, pra baixar o jquery precisa salva-lo como link, VQV !
